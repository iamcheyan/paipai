## 砍 scope 触发条件

出现以下情况 **立刻砍功能**，保 demo：

1. 距彩排还有 2 小时，happy path 还不通
2. 依赖的外部 API / 硬件现场不可用
3. 某功能预估还需 > 1 小时

**砍的顺序**: 边缘功能 → 真后端 → 换 mock → 换幻灯片一页带过
