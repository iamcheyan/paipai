## 选定方向

**拍拍** = 地点驱动的智能取景教练 + 拍照社交榜单

### 功能一：智能取景引导
用户在网红打卡点拍照时，APP 通过简单提示（文字 + 参考叠图）帮用户找到最佳角度和构图，轻松拍出大片。

### 功能二：地点拍照社交
调用 Google Maps API 绑定地理位置。类似大众点评的地点属性 + 照片榜单：
- 后来者选择同一位置，可看到前人上传的照片
- 支持点赞、留言
- 形成「此地最佳大片」排行榜
