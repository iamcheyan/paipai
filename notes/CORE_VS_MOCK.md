# 核心功能 vs 可 Mock 部分

**原则**: 明天只保证 **一条 happy path**：地图选点 → 看榜单 → 跟拍取景 → 拍照上传 → 榜单更新。

**最后更新**: 2026-06-26

## 技术栈决策（已定）

**Expo (React Native)** — 不做 Web PWA。

| 选项 | 开发速度 | 演示说服力 | 结论 |
|------|----------|------------|------|
| Web PWA | ⭐⭐⭐ 最快 | ⭐⭐ 浏览器里像「网页」 | ❌ 否决 |
| **Expo** | ⭐⭐ 一天可交付 | ⭐⭐⭐ 全屏相机 + 地图，像真 App | ✅ 采用 |
| 原生 Swift/Kotlin | ⭐ 太慢 | ⭐⭐⭐ | ❌ 黑客松不划算 |

```
Expo (React Native)
  ├── expo-camera            ← 全屏取景 + 叠图 overlay
  ├── react-native-maps      ← Google Maps（已开 SDK）
  ├── expo-router            ← 页面导航（地图→地点→相机）
  └── 本地 state + JSON      ← 社交 mock
```

**演示方式**：Expo Go 装评委/自己的手机 → 扫码即跑，不必上架。
